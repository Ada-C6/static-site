Ruby Projects

The projects listed here are classwork and homework projects I've worked on during my time at Ada Developers Academy, unless otherwise specified.

Farmer's Market ("FarMar")

Farmer's Market was an independent project where I learned how to:

  • Build a relational database using CSV files
  • Practice writing TDD specs
  • Practice using Ruby Enumerables

Scrabble

Scrabble was a pair-programming project where I learned how to:

  • Write TDD specs
  • Create Class and instance methods
  • Use constants
  • Work as both a driver and a navigator in a pair
  • Practice Git workflow with a partner

Random Menu Generator

One of our first projects using Ruby at Ada, my random menu generator creates fictional food menus inspired by music and Seattle. In this project, I learned about:

  • Using arrays and hashes
  • Using various loops
  • running code from the Terminal
