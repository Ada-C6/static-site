Finding great Coursera intro to programming options was pretty cool. Finding one with games as its projects? VERY cool.

I had so much fun in the class (at least when I wasn't pulling hair out over my latest broken code!) I definitely couldn't have done it without the mini-project development processes, class forum help, and e-mail exchanges with instructors/TAs.

But it was so satisfying to finish each game--I'm really glad that I stuck with it and didn't give up!
